============
Installation
============

This section explains how to install your local instance of plone intranet. Pick the quickstart option if you want to try it out. Pick the development option if you want to develop for Plone Intranet.

.. note::

    This project is still under construction and therefore you might run into many edge cases and side effects. It might be worth to look at the FAQ section in case you get an error.
